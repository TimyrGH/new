1.html

Является корневым элементом документа. Все остальные элементы содержатся внутри html.../html. Все, что находится за пределами элемента, не воспринимается браузером как HTML-код и никак им не обрабатывается.

2. title

Обязательным элементом раздела head является title. Текст, размещенный внутри элемента title, отображается в строке заголовка веб-браузера.

3. h или же подзагаловки.

Всего существует шесть тегов заголовков HTML:

- заголовок первого уровня(h1);

- заголовок второго уровня(h2);

- заголовок третьего уровня(h3);

- заголовок четвертого уровня(h4);

- заголовок пятого уровня(h5);
- заголовок шестого уровня(h6).

4.font редактор текста

Текст с размером 8, синий цвет, шрифт калибри, этот текст был изменен с помощью тега font.

5.изображения

Тег img используется для вставки графического изображения (картинки) в HTML документ. HTML тег img имеет два обязательных атрибута: src - адрес файла картинки и alt - альтернативный текст, который будет отображен, если картинка не может быть загружена. Также не лишним будет использование глобального атрибута title.
